## 备注说明

- **fusion 字段**：含 `fusion` 的物品为合成配方物品，需消耗指定材料合成。
- **rank 值**：部分区间中 `-5` 为顶级（无上/仙阶极品），数值规则在不同系统中并非完全线性。
- **type 字段**：
  - 技能类通常为 `type="技能"`（功法/辅修/神通/身法/瞳术）。
  - 装备类通常为 `type="装备"`（法器/防具）。
  - 饰品为 `type="饰品"`。
- **技能五类**共享技能体系，通过 `item_type` 细分。
- **炼丹相关**：合成丹药含 `elixir_config`，炼丹炉提供额外产出加成。
- **礼包特性**：后段礼包（如 `18130+`）出现 `roll_pool` 随机抽取结构与 `quality` 品质字段（主要用于饰品礼包）。
- **宠物特殊物品**：
  - `20027-20031` 为一至五阶天地灵髓，可用于宠物喂食并增加宠物经验；阶数限制为对应 ★ 级及以下宠物，`1★ = 5☆`。
  - `20032` 为启明石，用于宠物启明，重新随机获得 1 个基础技能。
  - `20033-20037` 为宠物蛋，使用后获得指定稀有度宠物：常见、普通、卓越、传说、神话。
  - 地图奖励中宠物蛋、天地灵髓、启明石跟随饰品礼包同一外层概率掉落；太初灵宠蛋、启明石、四阶/五阶天地灵髓在宠物资源池内权重较低。
- **boss神通**：`data/xiuxian/功法/boss神通.json` 有 `14001-14032` 共 32 条，供战斗逻辑单独读取；当前不在 `Items.export_items_data()` 的普通物品表中，且 `14001-14014` 与身法 ID 重叠。
