
# 修仙世界物品系统说明

## 📋 物品类型总览

修仙世界中的物品系统包含以下主要类型：

| 物品类型 | 说明 | 代表物品 |
|---------|------|---------|
| **功法** | 提升基础属性和修炼速度的主修功法 | 吐纳功法、太乙玄功 |
| **辅修功法** | 提供特殊技能加成的辅助功法 | 饮血术、斗战诀 |
| **神通** | 战斗中使用的特殊技能 | 灵光印、封神剑 |
| **身法** | 提升闪避率和命中率 | 八步赶蝉、混沌无相 |
| **瞳术** | 提升命中率和特殊视野能力 | 异瞳、鸿蒙始瞳 |
| **防具** | 提供防御属性加成 | 修士道袍、轮回魔铠 |
| **法器** | 提供攻击属性加成 | 精铁符剑、龙渊剑 |
| **丹药** | 恢复状态、增加属性、提升突破概率 | 生骨丹、天命血凝丹 |
| **药材** | 炼丹的基础材料 | 恒心草、太乙碧莹花 |
| **合成丹药** | 通过炼丹获得的强化丹药 | 洗髓丹、太上玄门丹 |
| **炼丹炉** | 提升炼丹产出数量 | 寒铁铸心炉、太上造化炉 |
| **聚灵旗** | 提升洞天福地灵气汇聚速度 | 一级聚灵旗、无上聚灵旗 |
| **神物** | 提供大量修为的珍贵物品 | 薤叶芸香、造化玉碟 |
| **特殊物品** | 具有特殊功能的道具 | 秘境钥匙、祈愿石、合欢玉 |
| **礼包** | 包含多种物品的组合包 | 新手大礼包、终极大礼包 |

## 🔮 功法系统

### 功法属性说明
- **hpbuff**: 提升气血百分比
- **mpbuff**: 提升真元百分比
- **atkbuff**: 提升攻击力百分比
- **ratebuff**: 提升修炼速度百分比
- **crit_buff**: 提升会心率百分比
- **def_buff**: 提升减伤率百分比
- **critatk**: 提升会心伤害百分比
- **two_buff**: 增加双修次数
- **number**: 提升突破概率百分比
- **dan_exp**: 提升炼丹经验
- **dan_buff**: 提升炼丹产出丹药数量
- **reap_buff**: 提升药材收取数量

### 功法等级划分
1. **人阶下品/上品** - 基础功法，适合初入修仙
2. **黄阶下品/上品** - 中等功法，提供较好属性加成
3. **玄阶下品/上品** - 高级功法，属性显著提升
4. **地阶下品/上品** - 顶级功法，接近仙道
5. **天阶下品/上品** - 仙道功法，属性极为强大
6. **仙阶下品/上品/极品** - 仙界功法，超越凡人
7. **无上** - 至高功法，接近天道

## ⚔️ 神通系统

### 神通类型说明
- **1-攻击** - 单次或多次攻击技能
- **2-持续伤害** - 造成持续伤害效果
- **3-增强自身** - 提升自身属性
- **4-封印** - 封印对手
- **5-随机伤害** - 随机攻击敌人
- **6-叠加伤害** - 伤害逐次叠加
- **7-变化神通** - 随机使用多种技能

### 神通属性说明
- **atkvalue**: 攻击倍数
- **turncost**: 释放技能后的回合休息
- **rate**: 释放概率百分比
- **hpcost**: 消耗当前血量的百分比
- **mpcost**: 消耗真元的百分比
- **success**: 封印技能的命中成功率百分比

## 🏠 洞天福地系统

### 洞天福地功能
- **修炼加速**: 提升修炼速度
- **灵田管理**: 种植药材，每23小时可收获
- **灵气汇聚**: 通过聚灵旗提升灵气浓度

### 聚灵旗等级
1. **一级聚灵旗** - 修炼速度+50%，药材速度+50%
2. **二级聚灵旗** - 修炼速度+100%，药材速度+100%
3. **三级聚灵旗** - 修炼速度+150%，药材速度+150%
4. **四级聚灵旗** - 修炼速度+200%，药材速度+200%
5. **仙级聚灵旗** - 修炼速度+250%，药材速度+250%
6. **无上聚灵旗** - 修炼速度+300%，药材速度+300%
